## Latest Polls

Displays latest US election polls using [FiveThirtyEight](https://projects.fivethirtyeight.com/polls/polls.json) polls data. 

[Live Demo](https://latest-polls.herokuapp.com/)

![](latestpolls_demo.gif)


## How to run 
1. Clone repository
2. `npm install`
3. `npm start`
